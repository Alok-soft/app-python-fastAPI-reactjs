# app-python-fastAPI-reactjs

A simple React + Vite authentication application with:

- User Sign Up
- User Sign In
- JWT Token Authentication
- Password Validation
- Toast Messages
- FastAPI Backend Integration
- LocalStorage Token Management

---

# Tech Stack

## Frontend
- React.js
- Vite
- React Router DOM
- CryptoJS

## Backend
- FastAPI
- JWT Authentication
